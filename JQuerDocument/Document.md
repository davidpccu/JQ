# jQuery Document

## different with document.ready

+ $(function)
```
$(function() {
    //dosomething
});
```
+ (function($))(jQuery)
```
(function($){
    //something
})(jQuery);
```


##Animates
Syntax: 
<pre><code>.animate( properties [, duration ] [, easing ] [, complete ] )</code></pre>
EX.
<pre><code>
$('#message').animate(
{  left: '650px',   opacity: 0.5,  fontSize: '24px'  },  1500
);
</code></pre>
